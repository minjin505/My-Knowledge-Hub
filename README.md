# My Knowledge Hub

A simple Personal Knowledge Management (PKM) web application built with HTML, CSS, and JavaScript.

This project allows users to create, manage, and organize personal notes in one place. It was created as a beginner-friendly project to learn Git, GitHub, and fundamental web development concepts.

## Features

- Create notes
- View saved notes
- Delete notes
- Simple and clean user interface

## Tech Stack

- HTML5
- CSS3
- JavaScript (Vanilla JS)

## Project Structure

```text
my-knowledge-hub/
├── index.html
├── style.css
├── script.js
└── README.md
```

## Getting Started

1. Clone the repository:

```bash
git clone https://github.com/your-username/my-knowledge-hub.git
```

2. Navigate to the project folder:

```bash
cd my-knowledge-hub
```

3. Open `index.html` in your browser.

## Learning Goals

This project was built to:

- Learn Git and GitHub workflows
- Practice HTML, CSS, and JavaScript
- Understand basic CRUD operations
- Build and maintain a personal project
